DOKI DOKI IF CONSTANTINE - README

1. DESCRIPTION DU PROJET

Ce projet est un Visual Novel narratif et psychologique centré sur l'ambiance intense des "Débatèques" universitaires. Le joueur incarne un nouveau participant qui doit interagir avec les autres personnages.

L'objectif est de naviguer entre la jalousie, la compétition et le chaos. Le scénario mène inéluctablement à une fin alternative violente basée sur les choix du joueur.

2. CONTEXTE ET PERSONNAGES

Le jeu prend place lors d'un débat sur la question : "La liberté commence-t-elle par le chaos ou par l’ordre ?".

* Antoine Torrens : Le directeur.
* Meha : Compétitive, observatrice.
* Anissa : Secrètement amoureuse et jalouse.
* Amir : Le "nerd" qui prévient des dangers.
* Fatiha : La gérante de la Débatèque.

3. DÉMARRAGE

Ce jeu utilise les technologies web standard.

1. Cloner ce dépôt Git.
2. Ouvrir le fichier index.html dans n'importe quel navigateur (Chrome, Firefox, etc.).

4. TECHNOLOGIES UTILISÉES

* HTML (Structure)
* CSS (Style)
* JavaScript (Logique du moteur de Visual Novel, gestion des scènes et des flags)
